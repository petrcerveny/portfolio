# Osobní portfolio – Petr Červený

## Popis projektu
Tento web slouží jako osobní portfolio, které prezentuje moji práci
z předmětů WWW (webové projekty) a DIM (grafická a video tvorba).

## Použité technologie
- HTML
- CSS
- GitHub Pages (hosting)

## Struktura webu
- index.html – úvodní stránka (o mně, kontakty)
- www.html – projekty z předmětu WWW
- dim.html – projekty z předmětu DIM

## Nasazení
Web je nasazen pomocí GitHub Pages.
Repozitář je veřejný a web je dostupný přes GitHub Pages z větve 'main'.

## Zdroje a licence
Veškeré obrázky, grafika a videa jsou vlastní tvorba vytvořená
v rámci školních projektů.
